AI 융합교육을 위한 인문사회 콘텐츠 설계

  • AIㆍ디지털 기술 및 원리와 사회과(일반사회, 역사, 지리, 윤리)의 구체적인 주제(인구)를 결합한 융합형 수업 콘텐츠 설계 능력을 함양한다.

  • 오픈소스 프로그래밍 언어(R), 마크다운 기반 디지털 출판 도구(Quarto), 서버형 웹 앱 개발 도구(Shiny)를 활용한 인터랙티브 교수ㆍ학습 웹 앱의 개발 역량을 함양한다.

  • 생성형 AI를 인터랙티브 교수ㆍ학습 웹 앱과 결합하는 다양한 방안에 대해 모색한다.
